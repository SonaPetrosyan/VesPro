

	
     Для предоставления информации о процессе приготовления пищи в пункты раздачи.
    

     Когда готовое блюдо принесут официанту, то
     установите флажок <Отдано> слева, чтобы удалить еду из списка.
    

      Если для параметра <Принтер> установлено значение 0, все блюда
      появиться в списке. В противном случае только блюда для данного
	  номера принтера будут в списке.
    
      При внесении каких-либо изменений нажмите кнопку <Редактировать>.
