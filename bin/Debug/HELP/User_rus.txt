
    Каждый сотрудник, вступающий в программу, имеет следующие свойства:

       - номер ресторана: <Restaurant>

       - номер зала: <Holl>

       и полномочия:

          — приемщик заказа: <orderer> + рабочее место: <workplace> (число – номер рабочего места)
            если эти поля отмечены , этот сотрудник может принять заказ

          — предыдущий получатель заказа: <previous> + рабочее место: <workplace>
            если эти поля отмечены , этот сотрудник может принять предварителныйзаказ

          — менеджер: <manager>
            если это поле отмечено, этот узел имеет право редактировать где угодно
            включая эту таблицу
 
          — редактор: <editor>
            если это поле отмечено, этот узел имеет право редактировать где угодно
            этой таблицы среди них нет.

          — наблюдатель: <observer>
            если отмечено только это поле, то этот сотрудник имеет право только наблюдать.
            нельзя редактировать

    Если у сотрудника нет ни одной из этих функций, он не может войти в программу.


    Эти разрешения устанавливаются с помощью кнопки <Сотрудники>.
  

    У каждого могут быть разные комбинации этих качеств.
  
 
    Принтеры, необходимые для печати заказа и чеков, определяются рабочим местом.
  
    Значение остальных полей следующее:

    <Password>: Пароль
    <Passqart>: пароль на магнитном носителе
    <Position>: Должность
    <Name>: Имя сотрудника


       Нажмите желтую кнопку <X>, чтобы закрыть.
