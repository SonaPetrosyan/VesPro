     Это список партнеров. 
	 Другими словами, список поставщиков и дебиторов.
	 Это также список кодов затрат.

     Для затрат отметьте столбец <затратa>.
     Они используются при работе с кнопкой 
	 <купить, переместить, продать, потратить>.

     Просмотреть совершенные с ними транзакции можно с
	 помощью кнопки <просмотр>, указав соответствующий код.

     Нажмите желтую кнопку <X>, чтобы закрыть.
